Raw SQL Blog Project (Batch 11)

Setup
1. Create a database from database.sql in phpMyAdmin.
2. Keep project in htdocs/blog_ostad.
3. Start Apache and MySQL from XAMPP.
4. Open in browser: http://localhost/blog_ostad

Features
- User registration with default user type: user
- Login and logout
- User dashboard with own blogs
- Create blog with title, short description, long description, image, publish status
- Update blog
- Publish and unpublish blog
- Blog details view
- Other users can comment on blog

Default database config
- Host: localhost
- Username: root
- Password: (empty)
- Database: raw_blog_batch11
